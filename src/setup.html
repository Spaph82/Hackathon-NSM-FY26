IB
Interactive Brokers
Live via TWS + local bridge
  1. In TWS: Global Configuration → API → Settings → enable "ActiveX and Socket Clients", port 7497 (paper), add 127.0.0.1 to Trusted IPs.
  2. Run the bridge: pip install ib_insync aiohttp then python ib_bridge.py.
  3. Keep TWS logged in and running while you trade.
V
VOLARB
Deribit · simulated · live data + AI engine
  1. The vol-arb engine paper-trades BTC options on real live Deribit data — no real orders, no keys needed.
  2. Run it: pip install -r engine/Hackathon-NSM-FY26/requirements.txt aiohttp then python deribit_bridge.py.
  3. Connect from the console — balance, positions, the AI's trades, and live P&L are all simulated off real prices.
